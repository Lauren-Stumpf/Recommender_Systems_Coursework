To run the UI enter the command python3 main.py in the command line and proceed from there. The following libraries will need to be installed 
os
tensorflow  (>version 2.7) 
numpy 
pandas

Python files: 
main.py - UI 
metrics_novelty_NDCG.py - metrics python file 
NeuMF_hybrid.py - 1st Recommender System 
RNN.py - 2nd Recommender System 
preprocessing_to_create_user..py - Preprocessing, feature extraction etc.
video - video

to run the models the library keras needs to be installed. 
They take quite a while to run so I have included models 
I trained on the NCC. The UI loads these models at runtime
